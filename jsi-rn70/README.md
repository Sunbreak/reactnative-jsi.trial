# react-native-jsi-rn70

JSI module with C++ for RN-v0.70.x

## Installation

```sh
npm install react-native-jsi-rn70
```

## Usage

```js
import { multiply } from 'react-native-jsi-rn70';

// ...

const result = await multiply(3, 7);
```
